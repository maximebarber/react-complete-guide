### Using Create React App

1. Install node.js

[node.js](https://nodejs.org/en/)

2. Install create-react-app

```sh
npm install -g create-react-app
```

3. Create a new project

```sh
create-react-app my-app
```

4. Start development

```sh
cd my-app/
```

```sh
npm start
```

Go to http://localhost:3000/

5. Deploy to production

```sh
npm run build
```
